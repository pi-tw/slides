Pandoc

COSCUP 2019

Peter Lu

跨平台文件格式轉換工具

支援格式

  • Markdown (including CommonMark and GitHub-flavored Markdown)
  • HTML5
  • EPUB version 2 or 3
  • LaTeX
  • Microsoft Word docx
  • OpenOffice/LibreOffice ODT
  • Jupyter notebook (ipynb)
  • MediaWiki markup
  • and more

指令

# 顯示支援輸入格式
$ pandoc --list-input-formats

# 顯示支援輸出格式
$ pandoc --list-output-formats

# 把 markdown 轉成 docx
$ pandoc -s -t docx test.md -o test.docx

# 把 markdown 轉成 revealjs html (簡報就是用這個做的)
$ pandoc -s --mathml -i -t revealjs test.md -o test.html
